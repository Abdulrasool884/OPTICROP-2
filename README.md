# OptiCrop – AI-Based Smart Crop Recommendation System

## Project Description

OptiCrop is a machine learning-based web application that recommends the most suitable crop based on soil nutrients and environmental conditions. The system helps farmers and agricultural stakeholders make informed crop selection decisions through an easy-to-use web interface.

## Features

- Crop recommendation using Machine Learning
- User-friendly Flask web application
- Data preprocessing and model training
- Real-time crop prediction
- Responsive web interface

## Technologies Used

- HTML
